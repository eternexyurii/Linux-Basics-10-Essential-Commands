# Linux-Basics-10-Essential-Commands
This repository shows 10 essential Linux commands with explanations and screenshots.   Each section contains:   1. **Command**   2. **Meaning**   3. **Screenshot**  
1. `ls`

Command: <img width="1920" height="1080" alt="ls" src="https://github.com/user-attachments/assets/9ab6c151-e787-457d-8c50-39c7e1d0e790" />
Meaning:
Lists files and directories in the current location.
2. cd

Command:<img width="1920" height="1080" alt="cd" src="https://github.com/user-attachments/assets/82a2bce2-7cd9-4c77-ae25-21ad0d694e65" />
Meaning:
Changes the current working directory to the given folder.
3. pwd

Command:<img width="1920" height="1080" alt="pwd" src="https://github.com/user-attachments/assets/87a68468-77b5-4d8d-8591-390537fba4a6" />
Meaning:
Prints the current working directory (shows where you are).
4. mkdir

Command:<img width="1920" height="1080" alt="mkdir" src="https://github.com/user-attachments/assets/fadb4fe4-6f57-475c-a79f-763bf0426ef1" />
Meaning:
Creates a new directory (folder).
5. rm

Command:<img width="1920" height="1080" alt="rm" src="https://github.com/user-attachments/assets/9719ca06-d3f2-444a-aeaa-5727a5a104fd" />
Meaning:
Removes (deletes) a file. Use rm -r foldername to delete a folder.
6. cp

Command:<img width="1920" height="1080" alt="cp" src="https://github.com/user-attachments/assets/f7e04b7c-f6cb-49f5-bd3e-e01a625a08c5" />
Meaning:
Copies a file to a new location or with a new name.
7. mv

Command:<img width="1920" height="1080" alt="mv" src="https://github.com/user-attachments/assets/37e9d527-c5d7-41e0-9891-68270b94b3aa" />
Meaning:
Moves or renames a file.
8. cat

Command:<img width="1920" height="1080" alt="cat" src="https://github.com/user-attachments/assets/feb0eb19-a9a2-4f02-9f86-b187ccedbee7" />
Meaning:
Displays the content of a file in the terminal.
9. touch

Command:<img width="1920" height="1080" alt="touch" src="https://github.com/user-attachments/assets/d1d84564-e1c4-473e-88b6-f16879a6c18d" />
Meaning:
Creates a new empty file (or updates the timestamp if it exists).
10. echo

Command:<img width="1920" height="1080" alt="echo" src="https://github.com/user-attachments/assets/c09ce22e-bf35-43eb-ab3e-9eb18398717b" />
Meaning:
Prints text to the terminal. You can also redirect output into a file:

echo "Hello" > hello.txt
